# E-commerce Dummy Text Generator (Figma Plugin)

이 플러그인은 이커머스 화면에서 자주 쓰는 **상품명**, **옵션명**, **가격**을 랜덤으로 생성해 선택한 텍스트 레이어에 바로 채워 줍니다. 각 레이어마다 서로 다른 문구가 적용되므로 상품 카드, 리스트, 상세 페이지 시안을 빠르게 구성할 수 있습니다.

## 주요 기능

- 세 가지 타입(상품명/옵션명/가격) 중 하나를 선택해 즉시 미리보기 확인
- `새 예시` 버튼으로 타입별 랜덤 텍스트 재생성
- 텍스트 레이어를 여러 개 동시에 선택하면 각각 다른 문자열이 자동 적용
- 선택 상태와 적용 결과를 패널에서 바로 확인

## 파일 구조

```
.
├── manifest.json   # Figma plugin manifest
├── code.js         # 텍스트 생성 + selection 적용 로직
├── ui.html         # 패널 UI와 사용자 상호작용
└── README.md
```

## 설치 및 사용법

1. **설치**
   - Figma 데스크톱 앱 → `Plugins` → `Development` → `Import plugin from manifest…`
   - 이 리포지토리의 `manifest.json`을 선택합니다.

2. **사용**
   - 변환할 텍스트 레이어(1개 이상)를 선택한 뒤 플러그인을 실행합니다.
   - `상품명 / 옵션명 / 가격` 중 원하는 항목을 고르면 예시 문구가 표시됩니다.
   - `새 예시`로 마음에 드는 랜덤 텍스트가 나올 때까지 돌려본 뒤 `선택된 텍스트 채우기`를 누르면 끝!

3. **주의/팁**
   - 플러그인이 텍스트를 수정하려면 해당 서체가 먼저 로드되어 있어야 합니다. 기본 폰트가 아니라면 한 번 수동으로 로드해주세요.
   - 가격은 최소 12,000원 ~ 최대 950,000원 범위에서 500원 단위로 생성됩니다.
   - 옵션 문자열은 색상/사이즈/구성/소재/특징 중 2~3개 조합으로 만들어집니다.

## 커스터마이징

- `code.js` 상단에 정의된 `BRAND_PREFIXES`, `STYLE_TAGS`, `PRODUCT_TYPES`, `OPTION_GROUPS`, `PRICE_RANGE`를 수정하면 회사 스타일에 맞는 텍스트 풀을 쉽게 확장할 수 있습니다.
- 패널 스타일이나 인터랙션은 `ui.html` 하나만 수정하면 되며, 추가 빌드 과정은 필요 없습니다.

필요한 문구나 로직을 자유롭게 확장해 보세요. 즐거운 디자인 작업 되세요! 🎨
