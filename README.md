# Korean Dummy Text Generator (Figma Plugin)

한국어 UI/UX 작업에서 자연스러운 문단과 문장을 빠르게 채워 넣을 수 있는 Figma/FigJam 플러그인입니다. 톤과 밀도를 조절해 상황에 맞는 더미 텍스트를 만들고, 선택한 텍스트 레이어에 바로 적용하거나 새 텍스트 레이어로 삽입할 수 있습니다.

## 주요 기능

- 잔잔한/친근한/기술 중심 3가지 톤 프리셋
- 문단 또는 단일 문장 모드, 최대 6개까지 생성
- 가볍게/보통/풍부하게 3단계 밀도
- 한 줄 헤딩 옵션, 즉시 복사 가능한 미리보기
- 선택한 텍스트 레이어 덮어쓰기 또는 새 텍스트 레이어 생성
- 선택 상태 실시간 표시, 적용 결과 상태 피드백

## 파일 구조

```
.
├── manifest.json   # Figma plugin manifest
├── code.js         # 플러그인 메인 스레드
├── ui.html         # 설정 패널 + 텍스트 생성 로직
└── README.md
```

## 설치 및 실행

1. **플러그인 로컬 설치**
   - Figma 데스크톱 앱 → `Plugins` → `Development` → `Import plugin from manifest…`
   - 이 리포지토리의 `manifest.json` 선택

2. **사용법**
   - 플러그인 실행 후 원하는 톤, 형식, 개수를 선택합니다.
   - `새로고침` 버튼으로 다른 조합의 텍스트를 만들거나, `복사`로 클립보드에 저장합니다.
   - 텍스트 레이어를 선택한 뒤 `선택된 텍스트 채우기`를 누르면 즉시 덮어씁니다.
   - 선택한 레이어가 없다면 `새 텍스트 레이어 만들기`로 뷰포트 중앙에 자동 배치합니다.

3. **테스트 팁**
   - 플러그인이 텍스트를 수정하려면 해당 서체를 먼저 로드해야 합니다. Figma 디폴트나 Noto Sans 계열은 자동으로 로드되지만, 커스텀 폰트는 먼저 로딩해주세요.
   - 문단 밀도는 문장 수를 조절해서 구성됩니다. 문장이 반복될 수 있으므로, 최종 레이아웃에서 적절히 변형해 사용하는 것을 권장합니다.

## 개발자 참고

- 번들러 없이 순수 HTML/JS로 구성되어 있어 바로 수정 후 리로드하면 됩니다.
- 텍스트 데이터는 `ui.html` 최상단의 `TEXT_BANK`, `HEADING_BANK`에서 추가/편집할 수 있습니다.
- 폰트 로딩 순서는 `code.js` 상단의 `DEFAULT_FONT_CANDIDATES`에서 정의합니다.

필요한 기능이 있다면 이슈로 남기거나 포크 후 원하는 문장 데이터를 확장해 보세요. 즐거운 디자인 작업 되세요!
